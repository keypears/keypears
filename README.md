# KeyPears

Password sharing social network.

- all data is always encrypted at rest
- users have a master key protected by a strong password
- keys are encrypted (again) with the user's master key
- a main instance is hosted at keypears.com
- users who don't want meta data shared with keypears.com can self-host
- primary function is sharing a key from user1@example1.com to
  user2@example2.com
