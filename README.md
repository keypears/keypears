# KeyPears

Social key manager.

## Features

- All data is always encrypted at rest
- Users have a master key protected by a strong password
- Keys are encrypted (again) with the user's master key
- A main instance is hosted at KeyPears.com
- Users who don't want metadata shared with KeyPears.com can self-host
- Primary function is sharing a key from user1@example1.com to
  user2@example2.com

## Apps

- Web app at KeyPears.com
- Self-hosted web app (with custom branding)
- Mobile app (iOS and Android)
- Browser extension (Chrome, Firefox, Edge)
- CLI app
- System tray app (Windows, macOS, Linux)

## Business Model

- Advertising
- Premium ("Verified") accounts
