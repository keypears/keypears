# About KeyPears

**Password manager for sovereign individuals**

KeyPears is a decentralized Diffie-Hellman key exchange platform that enables secure communication between any two email addresses (e.g., `alice@example.com` â†” `bob@example2.com`). Built on this foundation, it serves as both a password manager and cryptocurrency wallet that puts you in complete control while making it easy to sync across devices and share with others.

## Our Mission

We believe that managing passwords and secrets should be:

- **Secure by default** - End-to-end encryption with strong cryptography
- **User-controlled** - You own your data, not a corporation
- **Easy to use** - Simple, intuitive interface across all platforms
- **Open and transparent** - Open-source software and protocols

## The KeyPears Approach

Unlike traditional password managers and wallets, KeyPears uses decentralized Diffie-Hellman key exchange with an email-like architecture. You can:

- Establish secure communication channels with any email address across any domain
- Run your own KeyPears node for complete self-custody
- Choose from multiple service providers
- Share secrets securely with end-to-end encryption via DH key exchange
- Use it as both a password manager and cryptocurrency wallet
- Sync seamlessly across all your devices

## For Everyone

Whether you're a cryptocurrency enthusiast who needs secure wallet key management, a business team that needs to share credentials, or just someone who wants better control over their passwords, KeyPears is built for you.
